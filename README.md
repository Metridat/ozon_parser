# 🛒 Ozon Search Position Parser

Парсер для Ozon, который определяет позицию товара (SKU) в поисковой выдаче по заданному запросу.

---

## 📌 Описание проекта

Данный проект реализует асинхронный парсер поисковой выдачи Ozon с использованием Playwright.

Вместо парсинга HTML используется перехват внутреннего API Ozon, что делает решение:

- быстрее
- стабильнее
- менее зависимым от изменений верстки

---

## 🚀 Основные возможности

- 🔍 Поиск позиции товара по SKU
- 📡 Перехват внутреннего API (entrypoint-api)
- 🧠 Обход антибот-защиты
- 🔄 Поддержка прокрутки (infinite scroll)
- 📊 Сбор до 100 товаров за запрос
- 🧪 Режим стабильности (несколько запусков)

Проект использует следующие подходы для обхода антибот-защиты:

- playwright-stealth (обязательно)
- настройка user-agent и окружения браузера
- имитация поведения пользователя (задержки, скролл)
Прокси (опционально)

В случае получения ошибки 403 рекомендуется использовать SOCKS5 прокси:

python
proxy={"server": "socks5://127.0.0.1:10808"}
